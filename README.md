# 🎮 Simon Game

A simple and interactive **Simon Game** built using **HTML, CSS, and JavaScript**.  
Test your memory by repeating the sequence of colors as it gets longer and more challenging with each level.

---

## 📌 Features
- 🎨 Interactive color-based gameplay    
- 📈 Increasing difficulty with each level  
- 🧠 Memory challenge game logic  
- 💻 Fully responsive design 

---

## 🛠️ Built With
- HTML5  
- CSS3  
- JavaScript 

---

## 🎯 How to Play
1. Press any key to start the game  
2. Watch the sequence of colors  
3. Repeat the sequence by clicking the buttons  
4. Each round adds a new color to the sequence  
5. If you make a mistake, the game is over
